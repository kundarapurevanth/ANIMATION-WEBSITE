# ANIMATION-WEBSITE
This project is a modern animated portfolio website built using HTML, CSS, and JavaScript, designed to showcase creative content with smooth, professional animations and an engaging UI.

. Project Overview
The website features a visually rich hero section with a gradient background, animated navigation bar, social media links, and dynamic content layout. All elements animate smoothly on page load, creating a premium and interactive user experience.

. Key Features

- Animated Navigation Bar

Smooth fade-down animation on load

Interactive menu links with hover effects

Gradient-based “Subscribe” button

- Hero Section Animations

Text animations using fade-up, zoom-in, and slide-left effects

Sequential animation delays for a professional entrance flow

Call-to-action “Watch Now” button with hover scaling

- Social Media Sidebar

Vertical social bar with Font Awesome icons

Slide-left animation for visual emphasis

Direct links to Facebook, LinkedIn, TikTok, and YouTube

- Responsive Content Layout

Flexbox-based left and right sections

Text content aligned with animated image card

Gradient-bordered image container for modern styling

- Animation System

Custom CSS keyframe animations

JavaScript triggers animations on page load

Reusable animation utility classes

Smooth transitions with opacity and transform effects

- Design Highlights

Radial gradient background for depth

Neon-style accent colors

Clean typography using Segoe UI

Minimal yet visually appealing UI

- Tech Stack

HTML5 – Structure

CSS3 – Styling, Animations, Layout

JavaScript (Vanilla JS) – Animation trigger logic

Font Awesome – Icons

- Why This Project Matters

Demonstrates strong understanding of CSS animations & timing

Shows clean UI/UX design sense

Perfect example of a creative frontend animation website

Ideal for portfolios, landing pages, and creative brands
